How to use JSON Formatter

1. Paste Code: Copy your raw JSON code into the left box.

2. Choose Action: Click "Format" to beautify the code (make it readable) or "Minify" to remove whitespace (reduce file size).

3. Copy Result: Copy the output from the right box.
